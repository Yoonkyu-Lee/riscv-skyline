# riscv-skyline

A bare-metal RISC-V "starry-night" screen saver — eight hand-written
assembly drawing primitives plotted onto a 640×480 RGB565 framebuffer
through QEMU's `bochs-display` PCI VGA, driven by an interrupt-paced
animation loop.
